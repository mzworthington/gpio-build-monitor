# Hardware

## Pin map

Default BCM pins (configurable via `pins` in `integrations.json`):

| Light | Default pin |
|-------|-------------|
| Green | 17 |
| Yellow | 18 |
| Blue | 22 |
| Red | 27 |
| Purple | 23 |

## Shopping list

Approximate cost from the [original build](https://mzworthington.co.uk/guides/i-built-a-build-monitor) (~£20):

| Item | Notes |
|------|-------|
| Raspberry Pi Zero | W model recommended for Wi-Fi |
| SD card | 8 GB+ |
| LEDs | 5 × 5 mm (green, yellow, blue, red, purple) |
| Resistors | One per LED |
| Frame | e.g. IKEA Ribba - houses the LEDs behind the glass |
| Switch | Power on/off |
| Jumper wires | Pi to breadboard or direct solder |

### Useful kit

- Anti-static band
- Soldering iron and solder (if not using a breadboard)
- Multimeter

### Where to buy

The [build guide](https://mzworthington.co.uk/guides/i-built-a-build-monitor#how-to-build-your-own) links to specific parts from The Pi Hut, Amazon, and IKEA.
